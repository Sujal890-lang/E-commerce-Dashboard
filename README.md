# 🛒 E-Commerce Dashboard

## 📌 Overview

The E-Commerce Dashboard is a data analytics and visualization project designed to provide insights into sales performance, customer behavior, product trends, and business growth. The dashboard helps stakeholders make data-driven decisions through interactive reports and key performance indicators (KPIs).

## 🚀 Features

* Sales Performance Analysis
* Revenue and Profit Tracking
* Customer Segmentation
* Product Category Analysis
* Monthly & Yearly Sales Trends
* Interactive Filters and Visualizations
* KPI Monitoring Dashboard

## 🛠️ Technologies Used

* Power BI
* SQL
* Microsoft Excel
* Data Cleaning & Transformation
* Data Visualization

## 📊 Dashboard Insights

* Total Sales Revenue
* Total Orders
* Profit Analysis
* Top-Selling Products
* Customer Purchase Trends
* Regional Sales Performance
